# 🏠 Ontario Housing Affordability Analyzer

## Overview

The **Ontario Housing Affordability Analyzer** is a full-stack web application that helps users analyze housing affordability across cities in Ontario. It combines housing market data, income statistics, and mortgage calculations to provide affordability insights through interactive dashboards and visualizations.

This project is designed to demonstrate skills in data analysis, software development, financial modeling, and data visualization.

---

# Features

* Mortgage & affordability calculator
* City comparison dashboard
* Housing affordability score
* Interactive Ontario map
* Housing price trend visualizations
* Rent vs. income analysis
